## 查看 node

```shell
$ nvm list
$ nvm list installed # 同 list, 查看已安装版本
$ nvm list available # 查看可安装版本号
```



## 安装 node

```shell
$ nvm install 版本
```



## 切换 node 版本

```shell
$ nvm use 版本
```



## 查看 node 版本

```shell
$ node v
```



## 卸载 node 版本

```shell
$ nvm uninstall 版本
```
